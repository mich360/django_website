djangoプロジェクト始め方 (プロジェクタ名:django_website)

コマンド
  1. django-admin startproject django_website
  2. cd django_website　　　
     cd /Users/naomichimasumoto/Desktop/django_website　作成後こちらから
  3. python3 manage.py runserver
説明
　　　　　
        pwd : 現在地コマンド　　　　
        django_website/setting内の　ALLONED_HOSTS = [] エラーは ["*"]とすると直る
        manage.py : 管理用コマンドファイル　　　　　　　
        setting とurlsの二つは必須
        アプリ作りコマンド : python3 manage.py starapp website 
        websiteに templates フォルダー作り

アプリ追加の仕方　最低限一つのアプリが必要

  1. 英数字とアンダースコア__でアプリ名決める　例 website
  2. python3 manage.py startapp website
  3. django_website/settings 内の INSTALLED_APPS = [~　最後に「'website',」と追加登録

プロジェクトプロジェクトは全体設定以外ほとんど触れない、
メインはアプリ、ひとつのプロジェクトプロジェクトに複数のアプリを追加できます。

webページの追加の仕方

websiteアプリ フォルダー内部作業

  1. templates フォルダにhtml追加する
  2. views.pyに返すビュー追加
  3. urls.pyでビューをURLに繋ぐ　全体設定のurls.pyでアプリに行く設定「include("website.urls")」
